# Weather Wire created by Ronald King, Pablo Garcia, Liam Bayley, and Miguel Angel Abadia

This app will give weekly forecast of the weather including hourly update. This app contains a location based API to give weather updates to user for constant up to date information. This appication works together with the national weather API (https://www.weather.gov/documentation/services-web-api). the database used to store our users location will be mongoDB compass 
search icon by - https://www.flaticon.com/free-icons/magnifying-glass. 
We began with the search bar component which links a search to grab the information from the API to grab the information to be placed on the page according to the the user is on. The nav bar was next to place the user in the correponding page which has updated information for user. The user of this application can also choose to log in and have the oppertunity to login with a user name and password to store the future information like varios locations. The weekly gives the user a five day forecast to inform the user of the temperature, humidity, wind speed and description.
