# Weather Wire created by Ronald King, Pablo Garcia, Liam Bayley, and Miguel Angel Abadia

This app will give weekly forecast of the weather including hourly update. This app contains a location based API to give weather updates to user for constant up to date information. This appication works together with the national weather API (https://www.weather.gov/documentation/services-web-api). the database used to store our users location will be mongoDB compass
